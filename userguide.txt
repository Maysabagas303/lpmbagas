#Maysa-Bagas-UKK-)2021

Instalasi WEB pada XAMPP,PHP,MySQL

1. install XAMPP
2. Ekstrak file Ujikom-Maysa-Bagas.rar
3. Pindahkan folder ke Xampp/htdocs
4. Jalankanalankan aplikasi xampp pada browser kalian dengan mengaktifkan Apache dan MySQL
5. Buka server PhHPMyAdmin
6. Buat Basisdata/Database
7. Import database yg tersedia dalam folder web yg di ekstrak tadi, lokasi database:
   (C:\xampp\htdocs\ujikom-maysa-bagas\database), yang bernama pengaduan_masyarakat.sql dan kirim/go
